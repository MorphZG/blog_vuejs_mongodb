# Personal Blog

## Scripts

```sh
npm install  #install dependancies
npm run dev  #compile and hot reload for development
npm run build  #compile for production
npm run lint  #lint with eslint
npm run format  #format with prettier
```

## List of features

- Rest API
- mongodb database
